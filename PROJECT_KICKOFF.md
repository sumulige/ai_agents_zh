## [Level 1] 项目本质

**AI Agents A-Z** 是 AI 时代的"菜谱大全"，旨在让每个人都能用 n8n 构建实用的 AI Agent。

一句话：用可视化的 n8n 工作流，零代码创建各种 AI 自动化 Agent，从视频生成到社交媒体管理。

> 💡 **够用了吗？** 如果你是项目干系人，只需了解项目本质，可以在此止步。
> 需要更多信息？继续阅读 [Level 2] 了解核心目标。

## [Level 3] 约束与边界

### 技术约束

```
┌─────────────────────────────────────────────────────────────┐
│                      技术边界                                 │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  核心工具: n8n (工作流自动化平台)                           │
│  主要语言: JavaScript/TypeScript (n8n 表达式)               │
│  部署方式: 本地/npm/Docker/n8n Cloud                        │

│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 非技术约束

| 约束类型 | 说明 |
|----------|------|
| **时间预算** | 每周发布 1-2 个新 Episode |
| **质量标准** | 每个工作流经过测试验证，文档使用分层解释法 |
| **团队规模** | 1 人主导 + 社区贡献 |

### 范围边界

```
包含范围:
- n8n 工作流模板和 JSON 文件
- 分层解释风格的文档
- YouTube 视频教程
- Skool 社区支持

不包含:
- 深度代码开发 (Python/JavaScript 等)
- 需要 SDK 的 AI 集成 (优先使用 API)
- 企业级部署支持

未来考虑 (Later):
- n8n 自定义节点开发
- 企业级 workshop
- 认证体系
```
